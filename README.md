## 📦 Deployment

This site is deployed via **GitHub Pages**:

1. Code is pushed to the `main` branch.
2. GitHub Pages is enabled under **Settings > Pages > Source = main**.
3. Live URL is generated: `https://<username>.github.io/<repo-name>/`
